# event-invite

**A party invitation written in plain language → a printable HTML invite.**

For parents, hosts, organizers, anyone planning an event. No tools
to learn beyond seven simple words.

## What you write

```
invite "Maya turns 6!"
    host "The Patel family"
    when "Saturday, June 14"
    time "2:00 - 5:00 pm"
    where "Lincoln Park, Pavilion 3"

    rsvp_by "June 7"
    rsvp_to "maya@example.com"

    note "There will be a unicorn cake and a butterfly hunt."
    bring "A book to add to Maya's library (any age)"
end
```

## What you get

A standalone HTML invitation card — pastel gradients, italic
headline, all the details cleanly laid out. Open in a browser to
print, or email it to guests as a `.html` attachment.

## Make it yours

Change the colors, fonts, or layout by editing `lib.capy` once.
Every invitation you ever write inherits the new design.
