# weekly-meal-plan

**A week of dinners written as a list → a printable meal planner.**

For parents and home cooks. Tape it to the fridge.

## What you write

```
week "March 10 - 16"
    serves 4

    monday    "Sheet-pan salmon with broccoli and lemon"
    tuesday   "Pasta with brown butter and sage"
    wednesday "Black bean tacos with avocado and lime"
    ...

    note "Buy fresh fish on Sunday or Monday for best quality."
end
```

## What you get

A clean weekly grid in green & white, with notes called out at the
bottom. Print it, share it via screenshot, paste into your family
calendar.

## Run

```sh
capy run lib.capy script.capy > week.html
open week.html
```
