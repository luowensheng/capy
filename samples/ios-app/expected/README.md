# Habit Tracker (iOS)

Generated by Capy.

## Screens

### Today's habits (`HomeView`)
- Daily checklist
- Streak counter
- Reset day

### Past 30 days (`HistoryView`)
- Calendar view
- Per-habit stats


## Open in Xcode

Create a new iOS project, then drag in the `Sources/` folder and
drop `Info.plist` into the project. Or build via SPM with
`swift build` (uses `Package.swift`).
