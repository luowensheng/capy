# Library composition in Capy
*By Capy contributor*

Tags: #dsl, #tutorial, #capy

---

> **Note:** This post is generated by a library that imports two others.
Capy libraries can import other libraries. The main library here imports common/types.capy and common/syntax.capy. The imported types and functions are merged into this library before it runs.
This is how a team keeps a single source of truth for things like Email validation, semver regex, or shared DSL keywords like 'tag' and 'meta'. Define them once; import them everywhere.
