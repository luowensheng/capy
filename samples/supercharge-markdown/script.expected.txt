# Adopting Capy at Acme

*By Alice Chen · 2026-05-24*

We replaced 3 hand-rolled code generators with a single Capy library. Here is what we learned.

## Why Capy

- One source file, multiple language targets — no drift.
- Library doubles as the spec; reviewers learn what's possible in 5 minutes.
- Generated output is reproducible byte-for-byte.
